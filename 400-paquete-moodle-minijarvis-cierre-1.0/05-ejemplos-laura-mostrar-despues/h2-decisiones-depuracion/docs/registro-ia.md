## 2. Uso 1

### Objetivo

```text
Entender por qué un comando en mayúsculas no funcionaba y cómo normalizar la entrada.
```

### Prompt o resumen fiel

```text
En Java estoy leyendo un comando con Scanner. ¿Cómo puedo hacer que AYUDA y ayuda se traten igual? Explícalo para una alumna principiante y no me hagas el programa completo.
```

### Resultado obtenido

```text
La IA explicó que podía usar toLowerCase() para pasar el texto a minúsculas y trim() para quitar espacios al principio y al final.
```

### Qué acepté

```text
Acepté usar scanner.nextLine().trim().toLowerCase() para guardar el comando normalizado.
```

### Qué modifiqué yo

```text
Lo integré en mi código y mantuve los comandos básicos de H2. No añadí memoria, clases nuevas ni IA real.
```

### Cómo lo verifiqué

```text
Probé ayuda, AYUDA y ayuda con espacios. Después comprobé que todos entraban en el comando ayuda.
```

### Riesgos detectados

```text
La IA podía proponer una estructura más avanzada con métodos o mapas de comandos. No la usé porque H2 debe centrarse en bucles y decisiones básicas.
```
