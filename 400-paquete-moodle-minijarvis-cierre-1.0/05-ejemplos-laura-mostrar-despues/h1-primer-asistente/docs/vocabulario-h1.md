Define con tus palabras los conceptos básicos de H1.

| Concepto | Definición con mis palabras | Ejemplo en mi código |
|---|---|---|
| Clase | Bloque principal donde se organiza el código Java. | `public class Main` |
| `Main` | Nombre de la clase principal del programa. | `Main.java` |
| Método `main` | Punto donde empieza a ejecutarse el programa. | `public static void main(String[] args)` |
| Variable | Lugar donde guardo un valor que puedo usar después. | `String userName` |
| Constante | Valor que no debería cambiar durante la ejecución. | `final String ASSISTANT_NAME` |
| `String` | Tipo de dato para texto. | `String userName` |
| `int` | Tipo de dato para números enteros. | `final int COURSE_YEAR = 2026` |
| `final` | Indica que el valor no debería cambiar. | `final String ASSISTANT_NAME` |
| `Scanner` | Herramienta para leer datos del teclado. | `new Scanner(System.in)` |
| `System.in` | Entrada estándar, normalmente el teclado. | `Scanner(System.in)` |
| `System.out.print` | Muestra texto sin saltar de línea. | Pregunta del nombre. |
| `System.out.println` | Muestra texto y salta de línea. | Mensajes del asistente. |
| Compilar | Convertir el código fuente en algo que Java puede ejecutar. | `javac src/Main.java` |
| Ejecutar | Poner en marcha el programa. | `java -cp src Main` |
| README | Documento que explica qué hace el proyecto y cómo ejecutarlo. | `README.md` |
