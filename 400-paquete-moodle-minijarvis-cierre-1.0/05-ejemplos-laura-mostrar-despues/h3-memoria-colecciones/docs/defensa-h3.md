## Preguntas esenciales

### ¿Qué colección has usado y por qué?

```text
He usado ArrayList<String> porque quiero guardar recuerdos en orden y mostrarlos después.
```

### ¿Cómo guardas un recuerdo?

```text
Leo el texto con scanner.nextLine().trim() y, si no está vacío, lo guardo con memories.add(memory).
```

### ¿Cómo muestras la memoria?

```text
Si memories está vacío muestro un aviso. Si no, recorro la lista con un for y muestro cada recuerdo.
```

### ¿Qué pasa si la memoria está vacía?

```text
El programa muestra No tengo recuerdos todavía.
```

### ¿Qué cambia con Python?

```text
En Python usaría una list y append(). En Java uso ArrayList<String>, add(), size() y get().
```
