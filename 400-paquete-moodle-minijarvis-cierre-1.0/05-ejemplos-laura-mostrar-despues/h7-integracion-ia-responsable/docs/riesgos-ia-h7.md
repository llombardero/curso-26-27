# Riesgos IA — H7

| Riesgo | Mitigación |
|---|---|
| Datos personales | `PromptSafety` bloquea palabras como DNI o teléfono. |
| Secretos/API keys | `.env` está ignorado y el simulador bloquea token/password/api_key. |
| Respuesta incorrecta | Se muestra aviso de validación humana. |
| Dependencia excesiva | El alumnado debe defender qué hace el simulador. |
