# Hitos del proyecto MiniJarvis

Esta carpeta agrupa los documentos específicos de cada hito del proyecto anual.

Estructura actual:

```text
hitos/
├── h0-torre-papel-scrum/
│   ├── 03-primera-semana-scrum-torre-papel.md
│   ├── 10-primera-sesion-presentacion-proyecto.md
│   ├── 11-guia-docente-torre-papel-scrum.md
│   └── 12-ficha-alumnado-torre-papel-scrum.md
└── h1-primer-asistente/
    ├── 13-guia-docente-h1-primer-asistente.md
    ├── 13B-ficha-alumnado-h1-primer-asistente.md
    └── 13C-checklist-correccion-h1.md
```

Criterio de organización:

- Los documentos generales del curso permanecen en la raíz.
- Los documentos específicos de cada hito se agrupan dentro de `hitos/`.
- Los ejemplos de alumna se mantienen en `99-ejemplos-alumna/` para no mezclar modelos de entrega con guías docentes.

Ejemplos relacionados:

```text

```
